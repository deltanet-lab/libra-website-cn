Libra Core实现了一个非中心化的，可编程的数据库，该数据库提供了一个可以赋能数十亿人的金融基础架构。

## 开发者注意事项
* Libra Core是一个原型。
* 我们的API在不断进化当中，旨在演示相关功能，预计在发布之前会有重大变化。
* 我们已经启动了一个测试网，它是Libra区块链软件早期原型的可用演示。
